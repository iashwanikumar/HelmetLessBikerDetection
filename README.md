Helmet-less Motorcyclist Detection System

This project is a deep learning-based system that detects motorcyclists and identifies whether they are wearing helmets using Faster R-CNN and YOLOv3 models.

Overview

Detects motorcyclists from images/videos
Classifies helmet vs no-helmet
Saves images of helmet-less riders
Generates annotated outputs

Tech Stack

Python
TensorFlow
OpenCV
NumPy
YOLOv3
Faster R-CNN

Project Structure
HelmetDetection/

├── output/

Usage

Clone the repo, install dependencies, and run:

python HelmetLessBikerDetection.py
Output

Processed images/videos with detections and saved helmet violation snapshots.
